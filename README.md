# claude-camp-w1-Practice

Money School Claude Camp — Week 1 Python 作业

## 练习内容

| 文件 | 功能 |
|------|------|
| 01_temperature_converter.py | 温度转换器（摄氏 ↔ 华氏）|
| 02_greeting.py | 个性化问候语 |
| 03_tip_calculator.py | 小费计算器 |
| 04_password_generator.py | 随机密码生成器 |
| 05_bmi_calculator.py | BMI 计算器 |

## 运行方式

```bash
python 01_temperature_converter.py
```

每个程序都包含输入验证和错误提示。
